# Selection Guide Page PRD
Terminal Reference & Selection Center

## 1. 背景与目标

基于厂家原始数据文档：
- `docs/20-selection-guide-table-data.md`

现有页面方案：
- `docs/20.5-slection-guide-page-plan.md`

本 PRD 目标：将方案收敛为可开发、可验收、可 SEO 上线的工程选型参考页。

页面要回答 3 个问题：
1. 这是什么端子类型（Type）
2. 绝缘类型与线径颜色如何匹配（Insulation + Color/Wire）
3. 螺柱和孔径如何匹配（Stud/Hole）

---

## 2. 路由与页面定位

- 推荐路由：`/selection-guide`
- 页面定位：工程选型参考中心（Reference Center），不是产品列表页。
- 页面类型：静态内容页 + 结构化表格 + 轻交互（锚点、折叠）。

---

## 3. 数据审计结论（区间异常）

结论：**区间异常主要集中在大尺寸尾段，可从主选型表剔除。**

### 3.1 已识别异常/高风险项（来自颜色线径表）

1. `96.30-117.2红 Bed`：`Bed` 明显为拼写异常（应为 `Red`）
2. `192.6-242.27`：与上下区间不连续，存在缺段疑点
3. `24.27-325.0`：疑似 `242.27-325.0` 的录入错误
4. `180 / 250 / 325` 行大面积空值
5. `60.57-76.28` 区间缺失（`1/0` 与 `3/0` 之间存在断档）

### 3.2 是否属于大尺寸末端

是。以上异常均位于高线径区段（AWG `3/0`、`4/0`、`300`、`400`、`600` 等附近），可归类为末端重载规格区。

### 3.3 主表纳入策略（本 PRD 固化）

- 主选型表仅纳入：**稳定区间数据**
- 颜色/线径主表保留范围：`0.10-60.57 mm²`（覆盖至 `AWG 1/0`）
- `>= 76.28 mm²` 的尾段数据默认不进主选型表
- 尾段数据处理：
  - 方案 A：不展示
  - 方案 B：附录区块展示并标记 `Data review pending`

默认执行：**方案 A（不展示）**

## 6. 三张主表定义

## 6.1 Table A: Terminal Type Reference

字段：
- Terminal Type
- Code
- Chinese Name（若原始有）
- Typical Use（可编辑说明）

规则：
- 以原始表中的 Type + Code 为准
- 缺失字段允许留空，不做主观补数

## 6.2 Table B: Insulation + Color/Wire Reference

Table B 分为上下两段（仍视为同一主表模块）：

### B-1 Insulation Type
字段：
- Crimp Type
- Code
- Insulation Material
- Temperature Rating

### B-2 Color & Wire Range（稳定区间）
字段：
- Color Code
- Wire Size Range (mm²)
- AWG
- Max Current (A)
- Notes

规则：
- 仅纳入 `0.10-60.57 mm²` 稳定区间
- 异常区间和空值区间不进入主表
- 颜色文本统一规范（`Red/Blue/Yellow/Black`）

## 6.3 Table C: Stud Size Reference

字段：
- Stud Size (inch)
- Stud Size (mm)
- Hole Size (mm)
- Hole Size (inch)

规则：
- 保留原始有效尺寸，覆盖至 `15/16"`
- 单位格式统一，保留 inch + mm 双单位
