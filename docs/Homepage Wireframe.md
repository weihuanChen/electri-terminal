Task:
Redesign the homepage for a professional industrial B2B product website.

Project context:
This website is a B2B industrial catalog website, not a SaaS landing page and not a generic brand site.
Main product categories include:
- Terminal Blocks
- Cable Glands
- Electrical Enclosures
- DIN Rail Accessories

Current issue:
The existing homepage feels too generic, too marketing-style, and not product-driven enough.
It lacks strong product category entry points, technical trust signals, and industrial structure.

Design goals:
1. Make the homepage feel like a professional industrial B2B website
2. Prioritize product discovery, category navigation, technical documentation, and RFQ conversion
3. Keep the page modular and reusable
4. Build a stable structure that can also support future product categories

Please redesign the homepage using this information architecture:

1. Top utility bar
- email / phone / language / quick business note

2. Main header
- logo
- products
- industries
- resources
- blog
- about
- contact
- search
- request quote button

3. Hero section
Left side:
- eyebrow label
- clear product-focused headline
- short supporting text
- category tags
- CTA buttons:
  - Browse Products
  - Request RFQ
  - Download Catalog

Right side:
- product collage or industrial application visual

Below hero:
- trust badges:
  - OEM / ODM
  - Technical Documentation
  - Fast RFQ Response
  - Global Supply
  - CE / RoHS Ready

4. Product categories section
- 4 to 8 category cards
- each card includes:
  - image
  - category name
  - short description
  - link

5. Featured product series section
- 3 to 6 series cards
- each card includes:
  - image
  - series name
  - category
  - quick specs
  - actions

6. Applications section
- industrial automation
- control panels
- power distribution
- machinery
- renewable energy
- rail / infrastructure

7. Why choose us section
- stable quality control

- fast sampling

- documentation support

8. Certifications and documentation section
- certifications area
- download cards
- support notes

9. Technical resources / blog section
- article cards with related product relevance

10. Bottom RFQ CTA section
- strong RFQ conversion block

Requirements:
- Use a clean industrial B2B style
- Avoid generic SaaS hero style
- Avoid overdecorated gradients and marketing visuals
- Make it feel structured, trustworthy, and product-oriented
- Prioritize readability and modular section design

Please provide:
1. Page structure explanation

4. A production-ready Next.js page/component structure
