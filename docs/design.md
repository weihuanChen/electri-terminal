Create a consistent design system for an industrial B2B electrical components website.

The website should resemble a professional industrial manufacturer catalogue rather than a SaaS platform or marketing landing page.

Reference style:
Industrial companies such as Phoenix Contact, WAGO, Weidmuller, and Schneider Electric.

The design should emphasize trust, engineering clarity, and product information.

GENERAL STYLE PRINCIPLES

• Clean and structured layout
• High information density
• Catalogue-style product presentation
• Minimal marketing language
• Professional and technical appearance
• Consistent components across all pages

Avoid:
• SaaS dashboard layouts
• startup landing page styles
• excessive card UI
• large marketing banners
• oversized typography
• excessive white space


COLOR SYSTEM

The color palette should follow an industrial design approach:

Primary background:
White (#FFFFFF)

Primary text:
Black / dark gray (#111111)

Secondary text:
Neutral gray (#6B7280)

Accent color (primary brand color):
Industrial blue (#1E3A5F)

Secondary accent color:
Industrial orange (#F97316)

Color usage rules:

• 80% black / white / gray
• 15% blue
• 5% orange

Use blue for:
• buttons
• active navigation
• product links
• highlighted titles

Use orange for:
• hover states
• small highlights
• icons
• visual emphasis

Avoid large colored backgrounds.
The overall appearance should remain clean and technical.


LAYOUT SYSTEM

Page maximum width:
1280px

Horizontal padding:
24px

Content layout for product pages:

Main content width:
72%

Sidebar width:
28%

Sidebar should be positioned on the right side and remain visible during scrolling.


HEADER DESIGN

Header should be simple, compact, and professional.

Structure:

Left:
Company logo

Center:
Main navigation

Products
Applications
Support
About
Contact

Right:
Search icon

Header height:
72px

Avoid large marketing elements in the header.


BREADCRUMB NAVIGATION

Breadcrumb should appear on all pages except the homepage.

Example structure:

Home > Products > Terminal Blocks

Home > Products > Terminal Blocks > Push-in Terminal Blocks

Home > Products > Terminal Blocks > Push-in Terminal Blocks > PT-2.5

Design requirements:

• small font size
• neutral gray color
• minimal visual weight
• simple ">" separator

Breadcrumb should resemble technical documentation navigation.

• an industrial product catalogue
• engineering documentation
• a professional manufacturer website

• a SaaS dashboard
• a startup landing page
