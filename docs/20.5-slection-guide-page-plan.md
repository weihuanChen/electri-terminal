# 页面定位

这页的作用不是展示资料，而是帮助用户快速回答 3 个问题：

1. **这是什么类型的端子？**
2. **它属于哪种绝缘/颜色体系？**
3. **它能匹配什么线径和螺柱尺寸？**

所以页面结构应该按这个决策顺序来。

---

**Terminal Type, Insulation & Stud Size Reference Guide**
